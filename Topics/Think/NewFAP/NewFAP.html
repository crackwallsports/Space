会计准则
规则约束, 原则和假设
准则
美国 : 公认会计原则 (GAAP : Generally accepted accounting principles)
国际 : 国际财务报告准则 (IFRS : International Financial Repor-ting Standards)
制定
美国 民间机构 : 美国财务会计准则委员会 (FASB : the Financial Accounting Standards Board, ）: 既负责制定会计通则也负责制定具体的会计准则
美国 政府机构 : 证券交易委员会 (SEC : Securities and Exchange Commission) : 负责为上市公司制定信息披露标准
国际 机构 : 国际会计准则理事会 (IASB : International Accounting Standards Board)
原则和假设 :
通用原则 : 指編制财务报表时所使用的各种基本假设、概念和指导方针, 从长期的会计实务中总结而来
四个基本原则
成本原则 : 以实际发生的成本为依据, 用现金或现金等价物衡量
收入确认原则 : 何时确认收入
确认 是指入账, 如果收人确认过早，就会高估企业的实际盈利能力。但如果收人确认过晚，则会低估企业的实际盈利能力
配比原则 : 必须记录用于创造收人所产生的费用
充分披露原则 : 详细拔露会影响使用者决策的隐藏在财务报表内部的信息
四个假设
持续经营假设 : 编制财务报表时要假定企业将会持续经营下去，不会关门或被卖掉
货币计量假设 : 能够使用货币单位来表示交易或事项
会计分期假设 : 经营过程可以被分为不同期间，诸如以月份或年份为期间，同时为这些期间出具财务报表
会计主体假设 : 指企业是独立于其所有者及其他企业之外的个体
具体原则 : 是指报告公司交易和事项时使用的各种具体原则
约束
实质性约束 : 财务报告仅需披露会对理性决策者的决策过程造成影响的信息
成本收益约束 : 披露该休息的收益必须大于其成本

会计等式
资产 = 负债 + 所有者权益
所有者权益 = 所有者-名下资本 - 所有者提取 + 收入 - 费用
资产 = 负债 + 所有者名下资产 - 所有者提取 + 收入 - 费用
等式 商业企业
销售净额 - 商品销售成本 = 毛利
毛利 - 费用 = 净利润
等式 服务性企业
收入 - 费用 = 净利润

账项调整
调整类型
预付账款 (待摊费用) : 指在获取收益前 预先支付的费用, 如 保险费, 物料, 其他
折旧 : 将 会随时间 使用价值逐渐降低(如逐渐磨损) 的资产 的成本 分摊到 各个 预期使用期中去 的过程
分摊方法 : 直线折旧法(平摊)
账面价值 (净额) : 资产成本 减 累计折旧 的 差额
预收账款 : 提供产品或服务之前 收到的现金
应付账款 (预提费用) : 某一会计期间已经发生, 但仍未支付和记录的 各种成本
应付工资, 应付利息, 应付租金, 应付税款
应收账款 : 某一会计期间已经实现, 但既没有记录 也没有收到现金 的 各种收入
会计期间 : 大多以 一年 为基本的会计期间
财务报表 反映的是 已经取得的收入 和 已经发生的费用
会计年度 : 任意连续的12个月
年度财务报表 : 报告期为一年的会计报告
中期财务报表 : 报告期 为 一个月,一个季度,半年
调整方法
权责发生制 : 确认 已经取得的收入 和 已经发生的费用
收付实现制 : 收到 现金时 确认 收入, 支出 现金时 确认 费用
遵循原则
收入确认原则
配比原则 : 将费用和其所带来的收入 计入同一个 会计期间
对财务报表的影响 (调整前)
预付账款 资产多记 所有者权益少记
预收账款 负债多记 所有者权益少记
应付账款 负债少记 所有者权益多记
应收帐款 资产少记 所有者权益少记
